# Compact "N types: A (5), B (2)" distribution string per group

Compact "N types: A (5), B (2)" distribution string per group

## Usage

``` r
hla_group_dist(t, k)
```

## Arguments

- t:

  A \[hla_group_tally()\] result.

- k:

  Number of groups.

## Value

Character vector of length \`k\`; NA where the group had no value.
