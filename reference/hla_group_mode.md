# Modal value per group (ties -\> alphabetically first)

Modal value per group (ties -\> alphabetically first)

## Usage

``` r
hla_group_mode(t, k)
```

## Arguments

- t:

  A \[hla_group_tally()\] result.

- k:

  Number of groups.

## Value

Character vector of length \`k\`; NA where the group had no value.
