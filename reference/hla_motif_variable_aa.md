# Residue(s) a sequence carries at the consensus "x" (variable) positions

Purely a display helper for node labels; never feeds clustering.

## Usage

``` r
hla_motif_variable_aa(seq, cons)
```

## Arguments

- seq:

  A CDR3 sequence.

- cons:

  Its cluster consensus string.

## Value

The residues at the variable positions, or "".
